Python ドキュメントについて

Python ドキュメントは、もともと Guido van Rossum 一人の手で 書かれたものですが、ここ数年ではだんだんと Python コミュニティ自体の 取り組みになってきています。まだまだ成長を続けているこの ドキュメント集は、様々な形式で利用でき、印刷向けにタイプセットされた PDF や PostScript のバージョンもあります。これらは Python Web サイト から入手できます。

貢献した人々の一覧です。

コメントや質問

ドキュメントに関する一般的なコメントや質問は、電子メールなら docs@python.org に送ってください。 ドキュメントに関して何か間違いを発見した場合には、 SourceForge.net の Python バグ追跡システムに報告してください。(訳注: 日本語ドキュメント に関するバグ報告は、 SourceForge.jp の Python ドキュメント 和訳プロジェクトのバグ追跡システム に報告してください。) 内容の誤りや説明不足な部分に対する改案、すでに利用できる機能の 説明を補完するような追加の文章をお持ちなら、ぜひご提供ください。 マークアップのことを気にする必要はありません; ドキュメンテーション チームが喜んで手配します。

このドキュメントに書かれている情報の使い方に関する質問は、 Python のニュースグループ、 comp.lang.python、または Python メーリングリスト (ニュースグループと橋渡しされていて、 同じ情報が流れます) に送ってください。

上で挙げたチャネルではいずれも、決して HTML メールを送らない ようにしてください。よろしくお願いします。
