\chapter{データの永続化}
\label{persistence}

この章で解説されるモジュール群は Python データをディスクに永続的な形式
で保存します。
モジュール \module{pickle} と モジュール \module{marshal} は
多くの Python データ型をバイト列に変換し、バイト列から再生成します。
様々な DBM関連モジュールはハッシュを基にした、文字列から他の文字列へのマッ
プを保存するファイルフォーマット群をサポートします。
モジュール \module{bsddb} はディスクベースの文字列から文字列へのマッピ
ングを、ハッシュ、B-Tree, レコードを基にしたフォーマットで提供します。
