\section{\module{pwd} ---
         パスワードデータベースへのアクセスを提供する}

\declaremodule{builtin}{pwd}
  \platform{Unix}
\modulesynopsis{パスワードデータベースへのアクセスを提供する
(\function{getpwnam()} など)。}

%This module provides access to the \UNIX{} user account and password
%database.  It is available on all \UNIX{} versions.
このモジュールは\UNIX{}のユーザアカウントとパスワードのデータベースへ
のアクセスを提供します。全ての\UNIX{}系OSで利用できます。

%Password database entries are reported as a tuple-like object, whose
%attributes correspond to the members of the \code{passwd} structure
%(Attribute field below, see \code{<pwd.h>}):

パスワードデータベースの各エントリはタプルのようなオブジェクトで提供され、
それぞれの属性は\code{passwd}構造体のメンバに対応しています(下
の属性欄については、\code{<pwd.h>}を見てください)。


\begin{tableiii}{r|l|l}{textrm}{インデックス}{属性}{意味}
  \lineiii{0}{\code{pw_name}}{ログイン名}
  \lineiii{1}{\code{pw_passwd}}{暗号化されたパスワード(optional))}
  \lineiii{2}{\code{pw_uid}}{ユーザID(UID)}
  \lineiii{3}{\code{pw_gid}}{グループID(GID)}
  \lineiii{4}{\code{pw_gecos}}{実名またはコメント}
  \lineiii{5}{\code{pw_dir}}{ホームディレクトリ}
  \lineiii{6}{\code{pw_shell}}{シェル}
\end{tableiii}

%The uid and gid items are integers, all others are strings.
%\exception{KeyError} is raised if the entry asked for cannot be found.

UIDとGIDは整数で、それ以外は全て文字列です。
検索したエントリが見つからないと\exception{KeyError}が発生します。

%\note{In traditional \UNIX{} the field \code{pw_passwd} usually
%contains a password encrypted with a DES derived algorithm (see module
%\refmodule{crypt}\refbimodindex{crypt}).  However most modern unices 
%use a so-called \emph{shadow password} system.  On those unices the
%field \code{pw_passwd} only contains a asterisk (\code{'*'}) or the 
%letter \character{x} where the encrypted password is stored in a file
%\file{/etc/shadow} which is not world readable.}

\note{伝統的な\UNIX{}では、\code{pw_passwd}フィールドはDES由来のアルゴリ
ズムで暗号化されたパスワード(\refmodule{crypy}\refbimodindex{crypt}モジュー
ルをごらんください)が含まれています。しかし、近代的なUNIX系OSでは\emph
{シャドウパスワード}とよばれる仕組みを利用しています。この場合には
\var{pw_passwd}フィールドにはアスタリスク(\code{'*'})か、\character{x}と
いう一文字だけが含まれており、暗号化されたパスワードは、一般には見えない
\file{/etc/shadow}というファイルに入っています。\var{pw_passwd}フィールド
に有用な値が入っているかはシステムに依存します。
利用可能なら、暗号化されたパスワードへのアクセスが必要なときには 
\module{spwd}モジュールを利用してください。} 

%It defines the following items:
このモジュールでは以下のものが定義されています:

\begin{funcdesc}{getpwuid}{uid}
与えられたUIDに対応するパスワードデータベースのエントリを返します。
\end{funcdesc}

\begin{funcdesc}{getpwnam}{name}
与えられたユーザ名に対応するパスワードデータベースのエントリを返します。
\end{funcdesc}

\begin{funcdesc}{getpwall}{}
パスワードデータベースの全てのエントリを、任意の順番で並べたリストを返し
 ます。
\end{funcdesc}
