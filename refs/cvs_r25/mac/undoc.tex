\section{\module{cfmfile} --- コードフラグメントリソースを扱うモジュール}
% Code Fragment Resource module
\declaremodule{standard}{cfmfile}
  \platform{Mac}
\modulesynopsis{コードフラグメントリソースを扱うモジュール}
% Code Fragment Resource module.

%% \module{cfmfile} is a module that understands Code Fragments and the
%% accompanying ``cfrg'' resources. It can parse them and merge them, and is
%% used by BuildApplication to combine all plugin modules to a single
%% executable.

\module{cfmfile} は、コードフラグメントと関連する``cfrg''リソースを処
理するモジュールです。このモジュールでコードフラグメントを分解やマージ
できて、全てのプラグインモジュールをまとめて、一つの実行可能ファイルに
するため、BuildApplicationによって利用されます。

\deprecated{2.4}{}

\section{\module{icopen} --- \method{open()}と Internet Config の置き換え}
% Internet Config replacement for \method{open()}
\declaremodule{standard}{icopen}
  \platform{Mac}
\modulesynopsis{\method{open()}と Internet Config の置き換え}
% Internet Config replacement for \method{open()}.

%% Importing \module{icopen} will replace the builtin \method{open()}
%% with a version that uses Internet Config to set file type and creator
%% for new files.

\module{icopen} をインポートすると、組込み \method{open()} を新しいファ
イル用にファイルタイプおよびクリエーターを設定するために Internet
Configを使用するバージョンに置き換えます。

\section{\module{macresource} --- スクリプトのリソースを見つける}
% Locate script resources
\declaremodule{standard}{macresource}
  \platform{Mac}
\modulesynopsis{スクリプトのリソースを見つける}
% Locate script resources.

%% \module{macresource} helps scripts finding their resources, such as
%% dialogs and menus, without requiring special case code for when the
%% script is run under MacPython, as a MacPython applet or under OSX Python.

\module{macresource} はスクリプトが MacPython 上や MacPython アプレッ
トおよび OSX Python 上で起動されている時、特別な処理をせずにダイアログ
やメニューなどのようなリソースを見つけるためのヘルパースクリプトです。

\section{\module{PixMapWrapper} --- PixMapオブジェクトのラッパー}
% Wrapper for PixMap objects
\declaremodule{standard}{PixMapWrapper}
  \platform{Mac}
\modulesynopsis{PixMapオブジェクトのラッパー}
% Wrapper for PixMap objects.

%% \module{PixMapWrapper} wraps a PixMap object with a Python object that

%% to and from \module{PIL} images.

\module{PixMapWrapper} はPixMap オブジェクトを Python オブジェクトでラッ

\module{PIL} 画像との相互の変換をするメソッドも用意されています。
