93--------------------
94
95In Python 3.0, there is only one integer type.  It is called :func:`int` on the
96Python level, but actually corresponds to 2.x's :func:`long` type.  In the
97C-API, ``PyInt_*`` functions are replaced by their ``PyLong_*`` neighbors.  The
98best course of action here is using the ``PyInt_*`` functions aliased to
t99-``PyLong_*`` found in :file:`intobject.h`.  The the abstract ``PyNumber_*`` APIs
t99+``PyLong_*`` found in :file:`intobject.h`.  The abstract ``PyNumber_*`` APIs
100can also be used in some cases. ::
