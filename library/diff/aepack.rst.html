12The :mod:`aepack` module defines functions for converting (packing) Python
13variables to AppleEvent descriptors and back (unpacking). Within Python the
14AppleEvent descriptor is handled by Python objects of built-in type
15:class:`AEDesc`, defined in module :mod:`Carbon.AE`.
