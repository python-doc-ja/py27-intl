\section{\module{cmp} ---
         ファイル間の比較}

\declaremodule{standard}{cmp}
\sectionauthor{Moshe Zadka}{moshez@zadka.site.co.il}
\modulesynopsis{ファイル間を効率的に比較する}

\deprecated{1.6}{\refmodule{filecmp} モジュールを代りに使ってください。}

\module{cmp} モジュールファイルの比較処理を行うための関数を定義
しています。この関数は高い効率で操作を行うためあらゆる近道を
取ります。

\module{cmp} モジュールでは以下の関数を定義しています:

\begin{funcdesc}{cmp}{f1, f2}
名前で与えられた二つのファイルを比較します。比較処理を最適化するために
以下のトリックが使われます:

\begin{itemize}
        \item タイプ、サイズ、最終変更時刻が同じファイルは一致すると仮定します。
        \item タイプとサイズの違うファイルは決して同じファイルではありません。
        \item このモジュールは、すでに比較したファイルはそのシグネチャ (タイプ、サイズ、最終変更時刻) が変更されるまで比較しません。
        \item 外部プログラムを呼び出しません。
\end{itemize}
\end{funcdesc}

例:

\begin{verbatim}
>>> import cmp
>>> cmp.cmp('libundoc.tex', 'libundoc.tex')
1
>>> cmp.cmp('libundoc.tex', 'lib.tex')
0
\end{verbatim}
