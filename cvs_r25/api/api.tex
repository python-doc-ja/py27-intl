\begin{abstract}

\noindent

このマニュアルでは、拡張モジュールを書いたり Python インタプリタを
アプリケーションに埋め込んだりしたい C および \Cpp{} プログラマ
が利用できる API について述べています。
\citetitle[../ext/ext.html]{Extending and Embedding the Python
Interpreter} は拡張モジュールを書く際の一般的な決まりごとについて
記述していますが、API の詳細までは記述していないので、
このドキュメントが手引きになります。

\warning{このドキュメントの現在のバージョンはまだ不完全です。
とはいえ、役に立つだろうと思います。引き続きドキュメントの整備を
継続して、 Python ソースコードのリリースとは別に、その時々で
新たなバージョンをリリースするつもりです。}

\end{abstract}
